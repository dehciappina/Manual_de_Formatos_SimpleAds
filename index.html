Especificações gerais

Regras e prazos para envios de material

O envio do material deve ser feito por e-mail, 72 horas antes da data de início da campanha, para validação da equipe. Caso o material seja reprovado, o prazo reiniciará.

Há especificações técnicas diferentes para cada formato, nas próximas páginas há o detalhamento de cada especificação.

Caso esteja utilizando o suporte criativo SimpleAds, forneça materiais suficientes para a criação, incluindo o briefing preenchido, PSDs, fontes e referências. O material deve ser validado pela equipe para que o prazo comece a ser contado.

Veiculação via tag

Aceitamos tags de agência para veiculação de formatos display e Rich Media. As seguintes macros são necessárias:

[countgo]: Macro para cliques;

[timestamp]: Macro para impressões.

Para formatos de vídeo, aceitamos tags VAST 3.0 e VAST 4.0.

É possível que a tag apresente um tempo de resposta inferior à veiculação da SimpleAds, o que pode causar discrepâncias no resultado. Nossa equipe analisa todas as campanhas servidas via tag para identificar o quanto antes essas possíveis discrepâncias e, se necessário, acionar o cliente e o adserver responsável pela tag para analisar a diferença e buscar uma solução para o problema. Como regra geral de mercado, discrepâncias de até 20% são consideradas normais.

Para controle de entrega, o relatório gerado pela SimpleAds é considerado, pois é gerado por um adserver internacional homologado.

Contabilização por terceiros

A contabilização pode ser feita através de adservers terceiros, utilizando pixel de contabilização de impressões e URL de redirecionamento com contabilização que devem ser enviadas junto com o material da campanha.

Uso de fontes

Para peças em HTML5, é permitido o uso de fontes externas importadas da biblioteca do Google Fonts.

Tempo de animação

Para peças em gif animado ou HTML5 não há restrição de tempo, porém é importante que a animação principal não ultrapasse 30 segundos, para elevar a retenção de atenção do usuário. O uso de looping na animação é opcional, dependendo exclusivamente da criação.

Para peças de vídeo, é altamente recomendável a utilização de filmes com cortes de 15s ou 30s.

Implementação de Clicktags e URLs de redirecionamento

Em todas as campanhas é obrigatório o envio de ao menos uma URL de redirecionamento, que leva o usuário à landing page da campanha, após o clique. Essa URL pode ser parametrizada para identificação do volume de entrega no Google Analytics.

Para peças em HTML 5, é necessária a inserção da clicktag abaixo:

1. Chamar a biblioteca da clicktag dentro do <head> da peça criativa:

<script src="//ns.sascdn.com/diff/templates/js/banner/sas-clicktag-3.1.js"></script>

2. Declarar a variável ‘clicktag’ e preencher com o valor, que é a URL de redirecionamento

<script>var clickTag = "https://www.url-de-redirecionamento.com";</script>

3. Criar uma área de clique para a peça, incluindo um ID nesse elemento, como:

<a id="clickArea"></a>

4. Através desse ID, é necessário associar o elemento à variável (este código deve ser chamado ao final do código, de preferência antes do fechamento da tag da peça criativa:

<script type="text/javascript">
   var clickArea = document.getElementById("clickArea");
   clickArea.onclick = function(){
   window.open(clickTag, "blank");
}
</script>

5. Salve o arquivo principal como index.html e envie todos os arquivos juntos em formato zip, sem subdiretórios.

Práticas vetadas

  • Não utilize imagens ou vídeos protegidos por direitos autorais sem autorização;

  • Não utilize scripts que desempenhem qualquer função além das necessárias para animação e/ou interação com a peça;

  • Campanhas podem ser vetadas ou submetidas à análise caso utilizem materiais ou URL de destino que possam caracterizar:

    • Coleta inapropriada de dados pessoais dos usuários;

    • Material político;

    • Materiais médicos ou de medicamentos controlados;

    • Fake ads e fake news ou qualquer outro tipo de mensagem inapropriada.

    Peças Rich Media com formulário de lead passarão por uma análise de nossa equipe de tecnologia.

    • Caso necessário, o prazo para aprovação dos materiais pode aumentar em 24 horas.

Peso máximo

O peso máximo de cada peça varia de acordo com cada formato, nas próximas páginas há a especificação de cada um. Procure deixar a peça com o menor peso possível, pois isso facilita o carregamento na página para os usuários e eleva o resultado da campanha. Utilize compressores de imagem, opte pelo formato mais leve para cada imagem ou priorize a utilização de sprites. Para códigos longos, minifique para reduzir o peso.

  • Para comprimir PNGs e JPGs, recomendamos Tiny PNG;

  • Para comprimir GIFs, recomendamos GIF Compressor;

  • Para minificar códigos JS, CSS e HTML, recomendamos Minify Code.

Compatibilidade

Nossos formatos são compatíveis com todos os navegadores, em todas as versões mais recentes que somadas atinjam mais de 99% dos usuários.
